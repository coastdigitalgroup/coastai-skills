# Secondary CTA Planner & Audit Checklist

Use this template to identify the best Secondary Call-to-Action (SCTA) for a
specific page and ensure it is implemented for maximum impact.

## 1. The Commitment Gap Audit

- [ ] **What is the Primary CTA?** (e.g., "Buy Now," "Schedule Demo")
- [ ] **What is the "Cost" of the Primary Action?**
  - [ ] Time (minutes/hours): ________
  - [ ] Money ($): ________
  - [ ] Data (fields required): ________
  - [ ] Privacy (phone/credit card required): ________
- [ ] **What is the current Bounce Rate of the page?** (If >60%, an SCTA is
      highly recommended).

## 2. Intent-Matching Strategy

Identify which SCTA best bridges the gap for your specific audience:

| If the user is looking for... | Best Secondary CTA |
| :--- | :--- |
| **Proof of Value** | "See the Results" (Case Studies) |
| **How it Works** | "Watch Product Tour" (Video) |
| **Personal Fit** | "Calculate My Savings" (Calculator) |
| **Technical Details** | "View Comparison Guide" (PDF/Chart) |
| **Low-Risk Interaction** | "Join the Newsletter" (Email) |

## 3. Implementation Checklist

### Visual Hierarchy
- [ ] **Is the SCTA visually subordinate?** (e.g., Ghost button vs. Solid primary).
- [ ] **Is the primary CTA still the "loudest" element?** (Squint test).
- [ ] **Are the two CTAs placed in close proximity?** (Decision grouping).

### Copy & Messaging
- [ ] **Is the SCTA benefit-driven?** (e.g., "See the tool in action" vs. "Learn
      More").
- [ ] **Does the copy emphasize low friction?** (e.g., "2-Min Video," "No CC
      Required").

### Tracking & Nurture
- [ ] **Is SCTA conversion tracked as a separate event?**
- [ ] **Is there a clear "Next Step" after the SCTA is completed?**
- [ ] **Can we identify users who took the SCTA and eventually converted to the
      primary?**

## 4. The "Micro-Commitment" Decision Matrix

Score your proposed SCTA (1-5, where 1 is bad and 5 is excellent):

| Criterion | Score |
| :--- | :--- |
| **Ease of Entry:** Can they do it in under 2 minutes? | |
| **Immediate Value:** Do they get an answer or insight right away? | |
| **Relevance:** Is it a logical step toward the primary goal? | |
| **Subordination:** Does it stay out of the way of ready buyers? | |
| **TOTAL SCORE** | **/20** |

*Target Score: 16+ for high-impact SCTAs.*
