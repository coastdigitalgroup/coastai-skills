Accessibility & Performance Highlights

  • Overlapping Native Inputs: Uses two overlapping native HTML5 <input type="range"> controls. This preserves native platform touch gestures and full, bulletproof keyboard support.
  • Keyboard Support out-of-the-box: Press Tab to focus on the min or max handles. Adjust with ← / → or ↑ / ↓. Jump using Home, End, PageUp, or PageDown.
  • Dynamic Stacking (Z-Index Swap): High-precision JavaScript swaps the z-index priority when a thumb is focused, hovered, or active. This ensures thumbs never become trapped or unselectable.
  • Screen Reader friendly mapping: Programmatically maps dynamic text labels using the aria-valuetext attribute so screen readers announce "$200" or "$800" instead of plain numbers.
  • Target Sizing: Thumbs are sized at 28x28px and the active track wrapper is 48px deep, creating generous interactive targets complying with WCAG 2.2 standards.
