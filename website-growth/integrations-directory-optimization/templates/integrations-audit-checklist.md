# Integrations Directory Audit Checklist

Use this checklist to evaluate the effectiveness of an integrations directory or app marketplace. A "No" in any category indicates a specific growth opportunity.

## 1. Findability & Navigation
- [ ] **Functional Categories:** Are integrations grouped by what they *do* (e.g., CRM, Analytics) rather than alphabetically?
- [ ] **Synonym Search:** Does the search bar return results for generic terms (e.g., "Chat") or synonyms (e.g., "SFDC" for Salesforce)?
- [ ] **"Request" Fallback:** If a search returns zero results, is there a clear "Request an Integration" or "Vote for this App" CTA?
- [ ] **Featured/Popular View:** Is the default view sorted by popularity or business value rather than a static list?

## 2. Decision Support (The Detail Page)
- [ ] **Outcome-Oriented Headlines:** Does the page title explain the *benefit* (e.g., "Automate your billing") instead of just the name?
- [ ] **Plan Compatibility:** Is it clear which plan the user needs to be on (e.g., "Requires Business Plan") to use the integration?
- [ ] **Visual Validation:** Are there screenshots or a video showing how the integration actually looks inside the product?
- [ ] **Permission Transparency:** Does the page explicitly state what data will be read, written, or shared?

## 3. Setup & Friction Reduction
- [ ] **One-Click OAuth:** Is the connection process handled via OAuth rather than requiring manual API keys?
- [ ] **Requirement Clarity:** Are external requirements (e.g., "Must be a Slack Admin") listed *before* the user starts the flow?
- [ ] **Step-by-Step Preview:** Does the user know exactly how many steps the setup will take (e.g., "3-minute setup")?
- [ ] **Error Fallback:** Is there a clear "Troubleshoot" or "Contact Support" link if the connection fails?

## 4. Post-Connection & Growth
- [ ] **Active State Indicator:** Once connected, does the directory show a clear "Connected" or "Active" badge on the integration card?
- [ ] **Configuration Shortcut:** Is there a "Settings" link directly in the directory to manage the existing connection?
- [ ] **Success Metric Visibility:** (Advanced) Does the UI show a data snippet confirming the sync is working (e.g., "Last synced 5 mins ago")?
- [ ] **Cross-Sell Opportunity:** Does the success page suggest *related* integrations (e.g., "Now that you've connected Slack, try our Jira integration")?

## 5. SEO & Metadata
- [ ] **Unique URLs:** Does every integration have its own indexable URL?
- [ ] **Title Tag Optimization:** Are title tags formatted for search intent (e.g., "[Product] + [Integration] - How to Connect")?
- [ ] **Breadcrumb Navigation:** Is there a clear path back to the main directory or category page?
