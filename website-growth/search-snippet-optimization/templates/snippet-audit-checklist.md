# Search Snippet Optimization Checklist

Use this checklist to audit and optimize individual pages that have high impressions but low Click-Through Rates (CTR).

## 1. Performance Baseline (Search Console)
- [ ] **Current Position:** Is the page in the top 10? (If not, focus on content first).
- [ ] **CTR Gap:** Is the current CTR below the average for its position?
- [ ] **Impression Volume:** Is the volume high enough to justify optimization?

## 2. Meta Title Optimization (The "Hook")
- [ ] **Keyword Placement:** Is the primary keyword in the first 3-4 words?
- [ ] **Length Check:** Is it between 50-60 characters? (Avoid truncation).
- [ ] **Benefit/Value Prop:** Does it include a reason to click (e.g., "Fast," "Free," "Award-winning," "2024")?
- [ ] **Brand Clarity:** Is the brand name included at the end?
- [ ] **Emotional Hook:** Does it use "power words" (e.g., "Complete," "Essential," "Ultimate")?

## 3. Meta Description Optimization (The "Pitch")
- [ ] **Reinforcement:** Does it expand on the title's promise?
- [ ] **Length Check:** Is it between 120-155 characters?
- [ ] **Call to Action (CTA):** Does it end with an invitation (e.g., "Get yours," "Read the guide," "Start today")?
- [ ] **Keyword Boldness:** Does it include primary and secondary keywords naturally?
- [ ] **Unique Content:** Is it specific to this page and not a generic "catch-all" description?

## 4. Rich Snippet Opportunities (The "Real Estate")
- [ ] **Product Schema:** (If e-commerce) Does it show Price, Stock, and Rating?
- [ ] **FAQ Schema:** (If informational) Can we add 2-3 helpful questions to occupy more space?
- [ ] **Review Schema:** Does the page have aggregate ratings that can be marked up?
- [ ] **Article/Recipe:** Is the most specific schema type being used?
- [ ] **Site Search:** Does the homepage have the `Sitelink Searchbox` markup?

## 5. Visual & Psychological Check
- [ ] **Stand Out:** Does the snippet look different from the results immediately above and below it?
- [ ] **Trust Signals:** Does it include trust indicators (e.g., "Since 1995," "10k+ Happy Customers")?
- [ ] **Freshness:** Does it mention a current year or recent update if applicable?
- [ ] **Mirroring:** Does it use the same language the user likely typed into the search bar?

## 6. Technical Validation
- [ ] **Mobile Preview:** Checked using a SERP simulator for mobile truncation.
- [ ] **Schema Validation:** Verified using the [Google Rich Results Test](https://search.google.com/test/rich-results).
- [ ] **Unique Titles:** Confirmed no other page on the site uses this exact title.
