Safe Area Aware Layout

This page demonstrates how to handle notches and dynamic viewport units.

Problem

Standard 100vh on mobile often causes content to be hidden behind browser toolbars, or fixed elements to overlap the home indicator.

Solution

Using 100dvh and env(safe-area-inset-*) ensures the layout stays within the visible, safe boundaries of the screen.
