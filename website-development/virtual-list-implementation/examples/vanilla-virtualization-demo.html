Under the Hood

  • Roving Tabindex: Tabbing into the viewport highlights the current virtual list item. Use the ArrowUp and ArrowDown keys to navigate; focus moves programmatically while preserving proper sequential accessibility loops.
  • ARIA Injection: Screen readers announce full list dimensions (e.g. "Item 4512 of 10000") even though only ~15 nodes physically exist in the DOM at any one time.
  • Layer Promotion: Check the developer tools' "Layers" panel. Note how only the virtual active container creates a compositor layer, avoiding CPU-heavy reflows.
