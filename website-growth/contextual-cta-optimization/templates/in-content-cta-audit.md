# In-Content CTA Audit & Mapping Template

Use this template to identify "dead-ends" in your existing content and map them to high-converting "Next Step" offers.

## Part 1: The Content Audit
For each high-traffic page, answer the following:

| Page URL | Primary Intent (Problem/Solution/Buyer) | Words per CTA | Sidebar Only? | Dead-End Finale? |
| :--- | :--- | :--- | :--- | :--- |
| /blog/example-1 | Problem | 1,200 (1 CTA) | Yes | Yes |
| | | | | |

**Audit Questions:**
- [ ] Does this page have at least one CTA per 500 words?
- [ ] Is the first CTA visible within the first 50% of the content?
- [ ] Does the "Conclusion" provide a clear path to action?
- [ ] Are CTAs visible on mobile (not hidden in sidebars)?

---

## Part 2: Intent-to-Offer Mapping
Match your content to the most relevant offer in your library.

| Content Awareness Level | User's Goal | Logical Next Step (CTA) |
| :--- | :--- | :--- |
| **Unaware / Problem Aware** | Understand the pain. | Checklist, Template, Cheat Sheet, "How-to" Guide. |
| **Solution Aware** | See how to solve it. | Case Study, Webinar, Feature Deep-Dive, Comparison. |
| **Buyer Aware** | Validate the choice. | Pricing Page, Free Trial, Demo Request, ROI Calculator. |

---

## Part 3: The Contextual Copy Checklist
When drafting your inline CTAs, ensure they meet these criteria:

- [ ] **Descriptive:** Does the anchor text describe exactly what the user will get?
- [ ] **Benefit-Driven:** Does it highlight the *result* (e.g., "Save time") rather than the *action* (e.g., "Click here")?
- [ ] **Natural:** Does it flow as a recommendation from the previous paragraph?
- [ ] **Scannable:** Is it visually distinct enough to be found by a "skimmer"?

---

## Part 4: Implementation Roadmap
1. **Identify Top 10 Pages:** Sort by sessions in your analytics.
2. **Perform Audit:** Fill out Part 1 for these pages.
3. **Select Offers:** Use Part 2 to choose the right CTA for each.
4. **Draft Copy:** Use Part 3 to write 2-3 variations per page.
5. **Implement & Test:** Add the CTAs and monitor CTR for 14 days.
