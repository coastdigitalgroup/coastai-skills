Accessibility Instructions & Keyboard Controls

  • Tab: Focuses the workspace tree (navigates into the currently active item). Tabbing again moves focus out of the entire sidebar.
  • Up / Down Arrows: Navigates through visible nodes vertically.
  • Right Arrow: Opens folders when collapsed. When already open, jumps to the first nested item.
  • Left Arrow: Closes folders when open. When already closed, moves focus up to the parent directory.
  • Enter / Space: Selects the active node. If a folder is targeted, toggles its open/closed state.
  • Home / End: Instantly focus the first item (root) or the last visible child node in the entire list.
  • Asterisk (*): Expands the currently focused folder and all of its sibling folders at the exact same depth level.
