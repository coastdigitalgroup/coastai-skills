# Codex Workspace Files

This directory contains OpenAI Codex support files for
`@coastdigitalgroup/coastai-skills`.

Claude Code remains the lead implementation agent and `CLAUDE.md` remains the
authoritative development guide. Codex uses these files for release readiness,
production stabilization, repository hygiene, changelog support, documentation
standardization, config cleanup, and review tracking.

## Files

- `change-watch.md` - lightweight branch, documentation, config, or
  release-candidate review log.
- `release-readiness.md` - Codex release, production stabilization, and handoff
  checklist.

## Related Root Files

- `CODEX.md` - Codex operating guide.
- `AGENTS.md` - shared agent operating guide.
- `COPILOT.md` - GitHub Copilot support-agent guide.
- `JULES.md` - Google Jules automated maintenance guide.
- `RELEASE.md` - package release checklist.
- `CLAUDE.md` - Claude Code implementation guide.
