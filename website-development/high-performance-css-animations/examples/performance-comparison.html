Animation Performance Comparison

Open Browser DevTools > Rendering and enable Paint Flashing to see the difference.

Note: Animations are paused because prefers-reduced-motion is active.
❌ Bad: Animating left
Mechanism: Triggers Layout -> Paint -> Composite.

Impact: The main thread must recalculate the geometry of the box and potentially its container every frame. High CPU usage, prone to jank.
✅ Good: Animating transform
Mechanism: Triggers Composite only.

Impact: The element is promoted to a GPU layer. Moving it doesn't affect document flow or require repainting. 60fps even on slow devices.
