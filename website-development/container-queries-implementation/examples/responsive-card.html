Container Queries vs. Media Queries

Standard media queries respond to the window size. Container queries respond to the space allocated to the component.

The Media Query Problem

Both cards below use @media (min-width: 800px). Because the screen is > 800px, they both try to be horizontal. The sidebar card is crushed.

The Container Query Solution

Both cards below are the exact same component. They use @container (min-width: 400px). The one in the sidebar "knows" it only has 300px and stays vertical.
