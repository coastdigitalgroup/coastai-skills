# Promotional Discount & Coupon Audit Checklist

## 1. Strategy & Profitability
- [ ] **Goal Alignment:** Is the discount's primary goal clear (Acquisition, AOV, or Inventory Clearance)?
- [ ] **Margin Protection:** Have you calculated the "break-even" point for the discount, including shipping and CAC?
- [ ] **Exclusions:** Are high-margin or new-arrival items excluded from deep discounts if necessary?
- [ ] **Stacking Rules:** Are rules set to prevent users from combining multiple high-value codes?

## 2. UI/UX & Interaction
- [ ] **Visibility:** Is the "Promo Code" field subordinate to the primary "Checkout" button?
- [ ] **Auto-Apply:** Do marketing links automatically apply the discount without requiring manual entry?
- [ ] **Validation Feedback:** Does the UI show "Code Applied: -$15.00" in green immediately upon entry?
- [ ] **Error Handling:** If a code is expired, does the error message suggest a current valid offer?
- [ ] **Persistence:** If a user applies a code and leaves the site, is the code still applied when they return?

## 3. Psychological Triggers
- [ ] **Threshold Reminders:** If using tiered discounts, is there a "Progress Bar" or "Nudge" in the cart?
- [ ] **Risk Reversal:** Is the discount value clearly subtracted from the subtotal *before* asking for payment info?
- [ ] **Urgency:** If the code is time-limited, is there a countdown or "Expires in X hours" notice in the cart?

## 4. Mobile Optimization
- [ ] **Keyboard Type:** Does the promo code field trigger a standard text keyboard (or numeric if applicable)?
- [ ] **Tap Targets:** Is the "Apply" button easy to tap on mobile (min 44x44px)?
- [ ] **Viewport Placement:** Is the discount summary visible above the fold on most mobile devices?

## 5. Measurement & Analytics
- [ ] **Redemption Tracking:** Are you tracking which specific marketing channels drive the most discount redemptions?
- [ ] **AOV Comparison:** Are you comparing the AOV of discounted orders vs. full-price orders?
- [ ] **Abandonment Correlation:** Are you measuring if clicking the "Promo Code" field correlates with session exit?
