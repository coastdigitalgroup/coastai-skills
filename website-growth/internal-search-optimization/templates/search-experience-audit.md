# Internal Search Experience Audit Template

Use this template to evaluate the effectiveness of a website's internal search
experience. Rate each item on a scale of 1-5 (1 = Poor/Missing, 5 = Excellent).

## 1. Discovery & Entry (The "Search Box" UX)

- [ ] **Visibility:** Is the search bar immediately visible without clicking an
      icon first?
- [ ] **Contrast:** Does the search bar stand out from the header background?
- [ ] **Placeholder:** Does it suggest specific queries (e.g., "Search for
      'Model X'...")?
- [ ] **Mobile Access:** Is there a dedicated search trigger in the mobile
      "thumb zone"?
- [ ] **Auto-focus:** On mobile, does the keyboard open automatically when
      search is triggered?

## 2. Assistance (The "Auto-Suggest" UX)

- [ ] **Speed:** Do suggestions appear within <200ms?
- [ ] **Visuals:** Are product thumbnails and prices included in suggestions?
- [ ] **History:** Are recent searches or popular terms shown to the user?
- [ ] **Categorization:** Are suggestions grouped by type (e.g., Products vs.
      Categories vs. Articles)?
- [ ] **Scannability:** Is the suggest dropdown easy to read on mobile?

## 3. Relevancy (The "Results Page" UX)

- [ ] **Top Match:** Is the #1 result the most logically relevant item?
- [ ] **Precision:** Are results filtered by "AND" logic (all words match)
      rather than "OR" (any word matches)?
- [ ] **Synonyms:** Does a search for a synonym (e.g., "couch") return the
      primary term (e.g., "sofa")?
- [ ] **Typos:** Does the system handle common misspellings (e.g., "vlevet" ->
      "velvet")?
- [ ] **Persistence:** Does the original query remain in the search box on the
      results page?

## 4. Recovery (The "Zero Results" UX)

- [ ] **No Dead-Ends:** Is the "No Results" page free of blank spaces?
- [ ] **Smart Fallback:** Are popular products or categories suggested?
- [ ] **Clarity:** Does the page clearly state what was searched for and why it
      might have failed?
- [ ] **Support:** Is there a clear path to contact support or use a different
      discovery method?
